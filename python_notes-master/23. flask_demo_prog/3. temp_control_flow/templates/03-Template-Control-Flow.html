Here we use a for loop for a list

    {% for pup in puppies %}
  • {{pup}}
    • {% endfor %}

We can also add if/else statements

{% if 'Scooby' in puppies %}

Found you in the list of puppies Scooby!

{%elif 'xyz' in puppies%}

Found you in the list of puppies xyz!

{% else %}

Scooby isn't in this list.

{% endif %}
